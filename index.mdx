## Welcome to IronRDP

IronRDP is a collection of Rust crates providing a comprehensive implementation of the Microsoft Remote Desktop Protocol. Built with security as a primary focus, IronRDP offers a modular architecture that enables developers to build RDP clients, servers, and custom protocol implementations.

## Key Features

<CardGroup cols={2}>
  <Card title="Modular Architecture" icon="cubes">
    Organized into tiers with strict quality standards and clear API boundaries
  </Card>
  <Card title="Multiple Codecs" icon="video">
    Support for RemoteFX, RLE, and RDP 6.0 bitmap compression
  </Card>
  <Card title="Virtual Channels" icon="broadcast-tower">
    Comprehensive static and dynamic virtual channel implementations
  </Card>
  <Card title="Security First" icon="shield-check">
    Built with security as a priority, including extensive fuzzing
  </Card>
  <Card title="Cross-Platform" icon="globe">
    no_std compatible with support for diverse platforms
  </Card>
  <Card title="Flexible I/O" icon="arrows-rotate">
    Both async and blocking I/O abstractions available
  </Card>
  <Card title="Client & Server" icon="server">
    Complete implementations for both client and server scenarios
  </Card>
  <Card title="FFI & WebAssembly" icon="bridge">
    Bindings for C/C# and WebAssembly support for browser integration
  </Card>
</CardGroup>
